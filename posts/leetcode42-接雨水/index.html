Leetcode42 接雨水

题目来源： leetcode 42题

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例 1：

输入：height = [0,1,0,2,1,0,1,3,2,1,2,1] 输出：6 解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 示例 2：

输入：height = [4,2,0,3,2,5] 输出：9

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/trapping-rain-water 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路上很简单，每个柱子的位置能够放下的最多的水是被该点左右两侧最大值限制。设$i$处能够放的最多的水为$f(i)$，左右两侧最高的柱子为$left, right$。那么有 $$f(i) = min[left-height(i), right-height(i)].$$

那么只需要每次遍历一遍$height$数组，对于每个$i$，再去搜索左右的最大值。时间复杂度$O(n^2)$，空间只需要每次遍历的时候更新一下最终结果就行，因而空间复杂度为$O(1)$。

class Solution {
public:
    int trap(vector<int>& height) {
        int l = 0, r = l + 1;
        int n = height.size();
        int res = 0;
        for (int i = 0; i < n; i++) {
            int leftMax = height[i];
            int rightMax = height[i];
            int maxHeightp = 0;
            for (int j =i-1; j >= 0; j--) {
                leftMax = (height[j] > leftMax) ? height[j] : leftMax;
            }
            for (int j =i+1; j < n; j++) {
                rightMax = (height[j] > rightMax) ? height[j] : rightMax;
            }
            maxHeightp = (leftMax-height[i] < rightMax-height[i]) ?leftMax-height[i]:rightMax-height[i];
            res += maxHeightp;
        }

        return res;
    }
};

提交之后很不幸TLE了，思考一下也正常，每个元素都要左右遍历一遍，其实做了很多的重复操作。每个点左右的最大值可以通过左右各自遍历一遍就可以得到，如此就可以将时间复杂度压缩到$O(n)$。本质上是一种动态规划算法。看了看题解，就是根据以上的思路利用两个变量去分别优化掉两个数组，可以把空间复杂度干到$O(1)$。不过对我来说，感觉没啥意义就不再搞了。

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        int res = 0;
        int leftMax[n], rightMax[n];
        leftMax[0] = height[0];
        rightMax[n-1] = height[n-1];
        int lMax = height[0], rMax = height[n-1];
        for (int i = 1; i < n; i++) {
            leftMax[i] = max(leftMax[i-1],height[i]);
            rightMax[n-1-i] = max(rightMax[n-1-i+1], height[n-1-i]);
        }
        for (int i = 0; i < n; i++) {
            res += min(leftMax[i], rightMax[i]) - height[i];
        }
        return res;
    }
};
